Bomma — test: vystavení zboží mimo Shopify

Kolekce Závěsná svítidla zobrazená na externí stránce dvěma způsoby. Tato stránka běží na GitHub Pages, tj. mimo Shopify.

Token si můžeš předvyplnit i přes URL: ?token=…. Token je veřejný (Storefront API), takže v URL nevadí.

Metoda A

Buy Button JS

Klasický embed přes sdks.shopifycdn.com/buy-button. Shopify vykreslí produkty do vlastního iframe/DOM, včetně košíku a přechodu do pokladny.

Zadej token a klikni „Načíst zboží".
Metoda B

Storefront Web Components

Novější HTML komponenty (<shopify-store>, <shopify-context>…). Renderují se přímo v našem HTML, plně stylovatelné přes CSS.

Zadej token a klikni „Načíst zboží".
